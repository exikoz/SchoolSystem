# What is "SchoolSystem"
This is a program which uses c# and a Sql. The user will have a menu which they can do multiple different things. 
Some of the things are adding a student to a course or a teacher to a classroom. 
If you select any of these they will be created through CRUD and saved in the database. 
From there you can enter the menu again and see what student has passed a course and who has failed the course.

# Info/requirments for running the program:
This program runs on 2 different programs:
- SQL Server Management Studio 2021
- Visual Studio 2022
- Build: .Net 
# What the program contains:
### An index
So that people cant have same email as eachother, these are shown in the database under "index".
